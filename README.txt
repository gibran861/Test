Algorithme sert a rendre la monnaie de maniere optimale(un nombre minimal de piece rendu pour n'import quel montant).
les pieces utilisé sont 2$,5$,10$.
#installation:
vous devez avoir PHP ,PHPInit,Composer,Xampp
Installer Composer :
	1.Consulter la page : https://getcomposer.org/download/ et télécharger "Composer-setup.exe"
	2.Exécuter « Composer-Setup.exe » pour installer "composer"
Installer PHPInit :composer require --dev phpunit/phpunit


##Executiondu Projet
1-placer le dossier sour le repertoire HTDOCS : C:././Xampp/Htdocs
2-Ouvrir le serveur Appache de XAMPP
3-tapez cette URL dans le navigateur:localhost/test/tes.php

##Tests

1.ouvrir l'invite de commandes dans le repertoire du projet 
2.tapez : phpunit MonnaieTest.php
